# Contributing to Inno

Thank you for your interest in contributing to Inno! To maintain a consistent and readable codebase, we enforce strict code formatting rules.

## Code Formatting

We use `rustfmt` to format our Rust code. All contributions must be formatted according to the project's `rustfmt.toml` before they can be merged.

### How to Format

To format your code, run the following command from the project root:

```bash
cargo fmt
```

### Verification

You can verify if your code meets the formatting requirements by running:

```bash
cargo fmt --check
```

This is especially important before submitting a pull request.
